# piを求める

m,y平面上に

        0 <= x <= 1
        0 <= y <= 1

の範囲に乱数で任意の点を作成する。

それが原点を中心とする半径1の円の中に収まってるかどうかを判別する。

すると、上記の範囲かつ単位園内に収まる確率は、以下の面積比で表せる。

        0 <= x <= 1
        0 <= y <= 1
        x^2 + y^2 <=1
と

        0 <= x <= 1
        0 <= y <= 1



そのため、乱数で求まった面積比から円周率を計算する。

        乱数の作成回数:10^13 [回]
        求まったpi: 3.1415928826844
        ググったpi: 3.14159265359

3.141592くらいまでは一致してますね。
